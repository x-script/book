##### 激活（linux)

```
vim ~/.bashrc

export WORKON_HOME=~/Envs	# virtualenvwrapper存放虚拟环境目录
source /usr/local/bin/virtualenvwrapper.sh
```

```
source ~/.bashrc	# virtrualenvwrapper会安装到python的bin目录下，所以该路径是python安装目录下bin/virtualenvwrapper.sh
```
